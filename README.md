# 📊 Financial Risk Analysis Dashboard (Power BI)

# 📌 Project Overview
# This project focuses on analyzing financial data from a loan service provider to gain insights into customer demographics, loan portfolio performance, and financial risk.

# The dashboard is built using Power BI and provides interactive visualizations to help understand customer behavior, loan distribution, and potential financial risks.

# 🎯 Objectives
# Analyze customer demographics (age, income, education, gender)
# Evaluate loan portfolio and performance
# Identify high-risk customers and default patterns
# Provide insights for better decision-making

# 🛠️ Tools & Technologies
# Power BI
# Power Query (Data Cleaning & Transformation)
# DAX (Measures & Calculations)

# 📂 Dashboard Structure
# 📄 Page 1: Customer Demographics
# -Total Customers
# -Average Age
# -Average Income
# -Customer distribution by:
# -Gender
# -Education Level
# -Credit Score analysis by Gender & Education
# -Interactive slicers:
# -Income Buckets
# -Credit Score Buckets

# 📄 Page 2: Loan Portfolio & Performance
# -Total Loan Amount
# -Average Monthly Installment
# -Interest Rate (Gauge Visualization)
# -Loan distribution by type
# -Loan status analysis (Active vs Defaulted)
# -Top 10 Active Loans
# -Top 10 Defaulted Loans

# 📄 Page 3: Financial Risk Analysis
# -Total Defaulted Loans & Amount
# -High Risk Loans & Amount
# -Risk categorization based on Credit Score:
# -High Risk (<580)
# -Moderate Risk (<670)
# -Low Risk (<740)
# -Very Low Risk (>740)
# Analysis by:
# --Employment Status
# --Income & Education
# --Credit Score Buckets

# 🔄 Data Transformation
# -Created Risk Category using conditional logic on Credit Score
# -Developed DAX measures:
# -Defaulted Loans
# -Defaulted Loan Amount
# -High Risk Loans
# -High Risk Loan Amount

# 📈 Key Insights
# Identified segments with high default rates
# Analyzed relationship between income, education, and risk
# Highlighted loan types contributing to higher defaults
# Provided clear visibility into customer credit behavior
